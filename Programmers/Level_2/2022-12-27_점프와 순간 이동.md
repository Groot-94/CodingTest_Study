# 점프와 순간 이동
## 문제 설명
- https://school.programmers.co.kr/learn/courses/30/lessons/12980

## 제출
```swift
import Foundation

func solution(_ n: Int) -> Int {
    var count: Int = 0
    var distance = n
    
    while distance != 0 {
        if distance % 2 == 0 {
            distance = distance / 2
        } else if distance % 2 != 0 {
            distance -= 1
            count += 1
        }
    }

    return count
}
```
