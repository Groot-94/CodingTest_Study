# H-Index
## 문제 설명
- https://school.programmers.co.kr/learn/courses/30/lessons/42747

## 제출
```swift
func solution(_ citations:[Int]) -> Int {
    let array = citations.sorted(by: <)
    var result = 0
    
    for index in 0...citations.count - 1 {
        let h = citations.count - index
        
        if array[index] >= h {
            result = h
            break
        }
    }
    
    return result
}
```
