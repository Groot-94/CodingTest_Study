# 멀리 뛰기
## 문제 설명
- https://school.programmers.co.kr/learn/courses/30/lessons/12914

## 제출
```swift
func solution(_ n:Int) -> Int {
    var result = [0, 1, 2]
    
    if n < 3 {
        return result[n]
    }
    
    for index in 3...n {
        result.append((result[index - 1] + result[index - 2]) % 1234567)
    }
    
    return result[n]
}
```
