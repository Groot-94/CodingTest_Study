# 약수의 합

## 문제 설명
- 정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

## 제한사항
- n은 0 이상 3000이하인 정수입니다.

## 제출
```swift
func solution(_ n:Int) -> Int {
    guard n != 0 else { return 0 }
    
    var result = 0
    let problem: [Int] = Array(1...n)
    
    result = problem
        .filter { n % $0 == 0 }
        .reduce(result, +)
    
    return result
1. }
```
