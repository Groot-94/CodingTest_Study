# 약수의 합
## 문제 설명
- 정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

## 제한사항
- n은 0 이상 3000이하인 정수입니다.

## 제출

```swift
func solution(_ n: Int) -> Int {
    var result = 0
    var count = 1
    
    while count <= n {
        if n % count == 0 {
            result += count
        }
        
        count += 1
    }
    
    return result
}
```
